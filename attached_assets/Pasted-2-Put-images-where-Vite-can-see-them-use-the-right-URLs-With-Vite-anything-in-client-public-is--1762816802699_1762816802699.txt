2) Put images where Vite can see them & use the right URLs

With Vite, anything in client/public is served at the site root (/).

Do this folder setup:

client/
  public/
    img/
      hero.jpg
      mechanic-residential.jpg
      van-setup.jpg

Then reference them like this (note the leading slash):

style={{ backgroundImage: "url(/img/hero.jpg)" }} 

and in your PhotoStrip or anywhere else: 

const items = [
  { src: "/img/mechanic-residential.jpg", alt: "OnSite mechanic at residential service" },
  { src: "/img/hero.jpg", alt: "Mobile service" },
  { src: "/img/van-setup.jpg", alt: "Mobile van with organized tools" },
];

Cache gotcha on iPhone: if the image still won’t change, add a tiny cache-buster once (then remove later

style={{ backgroundImage: "url(/img/hero.jpg?v=2)" }}
